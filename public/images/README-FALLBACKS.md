# Imágenes Fallback

Este directorio debe contener imágenes fallback optimizadas para cuando no hay imagen disponible.

## Imágenes necesarias:

1. `fallback-sucesos.webp` - Imagen genérica para noticias de sucesos
2. `fallback-deportes.webp` - Imagen genérica para deportes
3. `fallback-internacionales.webp` - Imagen genérica para noticias internacionales
4. `fallback-tecnologia.webp` - Imagen genérica para tecnología
5. `fallback-nacionales.webp` - Imagen genérica para noticias nacionales
6. `fallback-default.webp` - Imagen por defecto

## Especificaciones:

- Formato: WebP
- Tamaño: Máximo 150KB
- Dimensiones: 1200x675px (16:9)
- Calidad: 75-80%

## Cómo crear:

1. Descarga imágenes de Unsplash o usa imágenes propias
2. Usa Squoosh.app para optimizar
3. Configura:
   - Formato: WebP
   - Calidad: 75%
   - Resize: 1200x675
4. Guarda con el nombre correspondiente
